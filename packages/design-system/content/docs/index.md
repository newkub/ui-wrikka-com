# Introduction

Welcome to the documentation site built with Nuxt Content and UnoCSS! This site demonstrates how to create a VitePress-like documentation experience using Nuxt 3, without actually using VitePress.

## What is this?

This is a fully functional documentation site built with:

- **Nuxt 3** - The Vue.js framework for building modern web applications
- **@nuxt/content** - The official Nuxt module for writing content in Markdown, JSON, CSV and YAML
- **UnoCSS** - The instant on-demand atomic CSS engine

## Key Features

- **Markdown-based content** - Write documentation in Markdown with full syntax highlighting
- **Responsive design** - Works on all device sizes with a mobile-first approach
- **Dark mode support** - Built-in dark theme that's easy on the eyes
- **SEO-friendly** - Server-side rendered for optimal search engine optimization
- **Fast and lightweight** - Optimized for performance with minimal JavaScript
