# @wrikka/nuxt-design-system

Wrikka Design System - Nuxt-based component library

## Development

```bash
# Run dev server
bun dev

# Build
bun build

# Test
bun test

# Lint
bun lint
```

## Features

- Vue 3 Components
- Composables และ Utilities
- UnoCSS สำหรับ styling
- TypeScript support
- Vitest สำหรับ testing
